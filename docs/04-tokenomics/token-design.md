## 10. The Dual-Stake Model

This is CipherOcto's most powerful economic innovation.

### Core Concept

Every participant must stake **both**:

1. **OCTO** (global alignment)
2. **Role Token** (local specialization)

| Role | Stake Required |
| ---- | -------------- |
| Compute Provider | OCTO + OCTO-A |
| Bandwidth Node | OCTO + OCTO-B |
| Storage Provider | OCTO + OCTO-S |
| Orchestrator | OCTO + OCTO-O |
| Developer | OCTO + OCTO-D |
| Validator | OCTO + OCTO-N |

### Why This Matters

Dual staking solves three classic crypto failures simultaneously:

1. **Role Tourism** ‚Äî Participants jumping between roles chasing emissions

   Forces commitment; you must believe in your sector's economics

2. **Misaligned Incentives** ‚Äî Local optimization hurting global network

   OCTO stake protects ecosystem; role stake protects service quality

3. **Farm & Dump** ‚Äî Subtokens becoming temporary yield assets

   Continuous buy pressure across all tokens

### The Economic Geometry

- **OCTO** = Civilization layer (vertical alignment)
- **Role Tokens** = Professional guilds (horizontal specialization)

This mirrors real economies: national currency + professional licenses.

### Slashing Model

Two independent penalty systems:

1. **Global Slashing (OCTO)**
   - *Trigger:* Fraud, malicious coordination, protocol attacks
   - *Impact:* Ecosystem harmed

2. **Role Slashing (Subtoken)**
   - *Trigger:* Poor performance, SLA violations, downtime
   - *Impact:* Sector market harmed

Penalties hit exactly where risk exists.
